Projeto Docker

HTML + CSS + Apache

Sobre o Projeto

Este é um projeto de exemplo rodando em um container Docker com Apache.

O objetivo é demonstrar como servir arquivos HTML e CSS usando Docker Compose.

Tecnologias Utilizadas

  • Docker
  • Apache HTTP Server
  • HTML5
  • CSS3

Como Usar

  1. Clone o repositório
  2. Execute docker-compose up -d
  3. Acesse http://localhost
